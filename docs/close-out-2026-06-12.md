## What shipped

A static-first single page, six beats on one scroll, plus a live invitation form.

| Beat | Surface |
|---|---|
| 1 · Threshold | Hero: animated-glass film stand-in, wordmark, eyebrow, one headline, one ghost CTA |
| 2 · The idea | Manifesto — three house blocks (What it is / ends / keeps) |
| 3 · The numeral | Cormorant numeral, possession-rule copy |
| 4 · The glimpse | Three glass panels — The Summon / The Itinerary / The Invitation |
| 5 · CEIL | One Lora-italic line |
| 6 · The ask | 5-field form + unbundled consent + APP 5 notice |
| Footer | Wordmark · legal line · Privacy/Terms · contact@ |

Routes: `/` (static), `/privacy` (static), `/terms` (static),
`/api/request` (dynamic route handler), `/robots.txt`, `/sitemap.xml`.

## Stack

Next.js (App Router) + TypeScript · Tailwind v4 · self-hosted `next/font`
(Montserrat, Cinzel, Lora, Cormorant Garamond, Syncopate) · ESLint · Vitest.
No analytics, no trackers, no third-party SDKs — privacy is the architecture.

## Design fidelity

- Section styling ported character-for-character from the locked reference into
  `app/threshold.css`; palette tokens in `app/globals.css`.
- Copy reproduced verbatim, including the possession-rule numeral lines (you
  *hold* a numeral; founding members *hold* the first — never "known by"/"assigned").
- Type roles held: Cinzel letterspaced eyebrows · one Lora-italic CEIL line ·
  Cormorant numeral · Syncopate wordmark only · everything else Montserrat
  (Light for house blocks).
- Palette: navy `#031B28` the only true background; bronze `#B5996F` matte
  hairlines/accents (no gradients); off-white `#FAFAFA` text.

## Motion

- Reveals via IntersectionObserver (`components/Reveal.tsx`), threshold .18.
- Hidden start-state gated behind `html.js` (set pre-paint) → no-JS users and
  crawlers get fully-rendered content, no flash.
- Hero drift is pure CSS. `prefers-reduced-motion` stills drift and reveals
  everywhere; smooth-scroll falls back to auto.
- Hero film: ships as the animated-glass stand-in. The real loop is wired
  (`<video autoplay muted loop playsinline>` + poster + static mobile fallback +
  reduced-motion poster) behind `NEXT_PUBLIC_HERO_FILM_ENABLED`, default off.

## Form, data & compliance

- `RequestForm` → `POST /api/request` (JSON) → Supabase
  `public.invitation_requests` via the publishable (anon, INSERT-only) key.
- Server-side validation (`lib/validation.ts`, unit-tested): required
  first/last/email, length ceilings (100/100/320/100/200), email shape;
  country/referred_by optional → null when blank.
- **Consent unbundled, unchecked by default.** Submitting *without* consent is
  still accepted (consent governs marketing email only, not consideration) →
  `consent_marketing=false`, consent columns left null.
- **RLS contract honoured:** when consent is given, the row sets `consent_at`
  (now) and `consent_wording_version='threshold-2026-06-12'`; `status='received'`;
  `source='calena.com.au'`.
- Naive per-IP in-memory rate limit (`lib/rate-limit.ts`): 5 / 10 min / IP.
- Success copy locked: **"Your request is being considered."**
- APP 5 collection notice present beneath the form, linking to `/privacy`.

**Compliance audit:**
- ✅ APP 5 notice live · ✅ consent unbundled + recorded (with wording version) ·
  ✅ zero scarcity language (no caps/counts/timers/"limited") ·
  ✅ zero investment-return implication · ✅ ABN line present (placeholder).

## Accessibility pass

- Semantic landmarks: `banner` (header) / `main#main-content` / `contentinfo`
  (footer); each beat is a labelled `<section>`.
- "Skip to content" link (first focusable, hidden until keyboard focus).
- Visible keyboard focus via `:focus-visible` rings on links, inputs, checkbox,
  and CTAs; mouse focus stays quiet (aesthetic preserved).
- Decorative layers (`film` drift, glass placeholder lines) are `aria-hidden`.
- Success uses `role="status"`; form errors use `role="alert"`.
- **Intentional deviation from the reference:** the APP 5 notice text contrast
  was lifted from the reference's `rgba(250,250,250,0.38)` to `0.6` (~AA) — it is
  legally-required content and must stay readable. Logged here per the lock.
